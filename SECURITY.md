# Security Policy

This repository is a public interview exercise. It is not a production system.

## Please do not submit

- real credentials,
- real Twilio secrets,
- production API keys,
- private datasets,
- partner data,
- or undisclosed third-party secrets.

## Reporting

If you discover a problem that could expose credentials, private review criteria, or internal evaluation infrastructure, email `business@ravdevops.com` instead of opening a public issue.

## Scope note

This repo is designed for controlled interview use. Security controls here are educational and process-oriented, not a representation of RavDevOps production security posture.
